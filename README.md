# Virat Kohli Power BI Dashboard

This project is a Power BI dashboard analyzing Virat Kohli's career statistics
and performance trends across formats (ODI, Test, T20I).

## Tools Used
- Power BI

## Key Insights
- Runs scored by year
- Batting average and strike rate analysis
- Comparison of 4s and 6s
- Count of 50s and 100s

## Files in this Repository
- `VK Dashboard.pbit` – Power BI template file
- `Virat Kohli dashboard.pdf` – Exported dashboard view
